This is my personal custom dictionary and word list containing ~42,000 words. 

It is based on [12dicts](https://wordlist.aspell.net/12dicts/) 2+2+3frq, the top 1,000 words, plus my own words; the 26k.txt file with a word list I built from my writings and books. I also added shortcuts I pulled from the FUTO keyboard dictionary and some of my own. 

I have it here more as a backup than for anyone to use, but if you want to try it feel free, just know that for capitalized words that can be both cap and not I've kept the non-cap only.
