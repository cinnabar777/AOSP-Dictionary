These are formatted word Lists built from the levels, they are ready to compile.

The 200_shortcuts_only.txt file contains shortcuts pulled from FUTO keyboard. You will want to add some of them, especially if you want the letter "I" to be capitalized.

The number are approximately how many words that word list contains. 

The f= number was reduced to give greater distance between The too 100 words, which account for 50% of writing, the next 900 words, which account for 39% of writing, then all the rest, and to help separate the internally developed words from the dictionary.
